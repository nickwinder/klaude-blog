# Blog Post Template

Use this template for consistent blog post structure and formatting.

## Frontmatter Template

```yaml
---
title: "[Compelling, keyword-rich title 50-60 characters]"
description: "[Engaging meta description 150-160 characters with primary keyword and value proposition]"
pubDate: "[Month DD YYYY format - e.g., Aug 21 2025]"
heroImage: "../../assets/blog-placeholder-1.jpg"

draft: false

  canonical: ""
  noindex: false
  keywords: ["primary-keyword", "secondary-keyword", "long-tail-keyword"]

---
```

## Content Structure

### 1. Hook Introduction (150-200 words)

```markdown
[Start with a compelling statistic, surprising fact, or relatable problem]

[Brief context or problem statement]

[Clear value proposition - what readers will learn/gain]

[Preview of main points or structure]
```

### 2. Main Content Body (1200-1500 words)

```markdown
## [H2: Primary Topic Section - Include Keywords]

[Content with examples, explanations, and supporting evidence]

[Include code blocks, images, or visual aids where relevant]

### [H3: Subsection if needed]

[Detailed explanation or example]

## [H2: Second Major Section]

[Continue with logical flow and clear transitions]

[Internal links to related posts: [Related Article Title](link)]

[External links to authoritative sources: [Source Name](link)]

## [H2: Third Major Section]

[Actionable insights and practical applications]

[Real-world examples or case studies]
```

### 3. Strong Conclusion (200-300 words)

```markdown
## Key Takeaways

[Summarize main points in bullet format]
- [Key insight 1]
- [Key insight 2] 
- [Key insight 3]

## Next Steps

[Provide clear, actionable next steps for readers]

[Call-to-action for engagement - comments, shares, newsletter signup]

## Related Reading

[3-5 internal links to related content]
- [Related Post 1](link)
- [Related Post 2](link)
- [Related Post 3](link)
```

## Content Guidelines

### Writing Style
- Use active voice (80%+ of sentences)
- Vary sentence length for readability
- Include transitional phrases for smooth flow
- Write conversationally but professionally
- Include specific examples and case studies

### SEO Optimization
- Primary keyword in title and first paragraph
- Secondary keywords in H2 headings
- Natural keyword density (1-2%)
- Meta description with compelling CTA
- Alt text for all images

### Visual Elements
- Hero image that represents the topic
- Screenshots or diagrams where helpful
- Code blocks with proper syntax highlighting
- Bullet points and numbered lists for scannability
- Quote blocks for important insights

### Engagement Features
- Ask questions to encourage comments
- Include shareable quotes or insights
- Add practical checklists or templates
- Provide downloadable resources when relevant
- Reference community discussions or trends

## Quality Checklist

- [ ] Compelling headline with target keyword
- [ ] Engaging introduction with clear value prop
- [ ] Logical content structure with H2/H3 hierarchy
- [ ] 1500-2000 word count
- [ ] 3+ internal links to related content
- [ ] 2+ external links to authoritative sources
- [ ] Optimized meta description
- [ ] Relevant tags and categories
- [ ] Code examples properly formatted
- [ ] Images have descriptive alt text
- [ ] Strong conclusion with clear next steps
- [ ] Call-to-action for reader engagement
- [ ] Grammar and style consistency
- [ ] Mobile-friendly formatting
